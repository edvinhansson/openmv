# OpenMV
An open source metroidvania.

Created from the ground up in C99.

## Features
 - [x] Basic Rendering.
 - [x] Resource management.
 - [x] Code hot-reloading for game logic.
 - [ ] Physics.
 - [ ] Audio.
 - [ ] In-game GUI.
 - [ ] Debug IMGUI.
 - [ ] Save-game system.
 - [ ] Room system.
 - [ ] Tilemap system.
 - [ ] Advanced Rendering (Post processing fx, etc.).
